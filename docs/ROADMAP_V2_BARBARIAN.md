## 1. 게임 정체 — 새 정의

### 한 줄
> "작품 세계관에 빙의되어, 자연어로 지시하며, 사이드뷰 화면 + 채팅 묘사로 진행하는 시뮬 RPG"

### 핵심 흐름
```
1. 사용자 자연어 지시 (또는 선택지)
2. LLM이 가능성 평가 (★ "특이하고 말도 안 되는 선택지"도 평가)
3. 백엔드 시뮬 진짜:
   - HP/MP/스탯/위치 계산
   - 전투 / 이동 / 자원
   - 파티원 반응 (충성도/임기응변)
4. 사이드뷰 화면 갱신 (★ Phaser sprite)
5. 채팅에 묘사 (★ LLM)
6. 다음 턴 대기
```

### 차별화 (vs 경쟁)

| 항목 | AI Dungeon | Crack | FableAI | **WorldFork V2** |
|---|---|---|---|---|
| 시각화 | X | X | AI 이미지 | ★ 사이드뷰 sprite |
| 시뮬 게임 로직 | X | X | X | ★ HP/스탯/위치/전투 |
| 자연어 + 가능성 평가 | 약함 | 약함 | 보통 | ★ 강함 |
| 작품 충실도 | 약함 | 약함 | 약함 | ★ 1작품 깊이 |
| 게임감 | 텍스트만 | 텍스트만 | 텍스트+이미지 | ★ 진짜 RPG |

## 2. 작품 본질 (★ 바바리안)

> **상세 자료**: `docs/design/barbarian_extracted.md` (자료 진짜 인용 + 분석)

### 작품 정보
- 작가: 정윤강
- 작품: "게임 속 바바리안으로 살아남기"
- 장르: 게임빙의물 + 미궁물
- 시작: 2021-03-31 네이버 시리즈
- 인디 게임 "던전 앤 스톤"의 바바리안에 빙의 → 미궁 탐험

### 게임 시스템 본질 (★ 자료 진짜)

#### 정수 시스템 (★ 핵심)
- 몬스터 처치 → 0.01% 드롭
- 흡수 시 획득:
  * 액티브 스킬 1개
  * 패시브 스킬
  * 능력치 보너스 (★ 수십 개)
- 등급: 1(최강)~9(최약)
- 합성 정수 (★ 지하 1층 무지개섬 연구소):
  * A 장치 = 스탯
  * B 장치 = 패시브
  * C 장치 = 액티브
  * 마석 색깔 = 액티브 종류

#### 능력치 schema (★ 작품 진짜)

**일반 스탯**:
- 근력 / 민첩 / 유연성
- 시각 / 후각 / 인지력
- 명중률 / 회피
- 골강도 / 물리내성

**특이 스탯** (★ 본인 짚은 본질):
- 집착 — 정수 흡수 / 추적 / 끈기
- 육감 — 적중률 / 숨은 공간 / 위험 감지
- 항마력 — 마법 저항 (400+ = 제압 면역)
- 정신력 — 정신계 마법 / 상태 이상 저항
- 영혼력 (MP) — 레벨 업으로만 상승
- 인식방해 — 은신
- 인지력 — 위험 감지
- 지지도 — 우두머리 시 통솔력 / 정책 성공률

**빌드업** (★ CRPG 스타일):
- a/b/c 정수 조합 → d로 교체 시 재생력 잃고 거대화 탱커
- 발더스 게이트 / 패스파인더 스타일

#### 미궁 시스템
- 1~8+층 (★ 깊을수록 강함)
- 균열 (★ 난이도 +1, 8층 균열 = 9층 난이도)
- 차원문 (★ 7층부터 X)
- 영구사망 (★ HP 0 = 게임 끝)

#### 클래스
- 바바리안 (탱커/근접)
- 사제 (★ 부활 가능?)
- 연금술사 (★ 정수 합성)
- 탐험가 (★ 원거리 딜)

#### 캐릭터
- 비요른 얀델 (★ 주인공, 7등급 탱커 바바리안)
- 미샤 칼스타인 (★ 7등급 근접, 적묘족)
- 이한수 (★ 빙의 정보원?)

#### 세력
- 노아르크 vs 라프도니아
- 카루이

#### 화폐 (★ 자료)
- 마석 → 스톤 단위
- 9등급 마석 = 20스톤
- 5등급 정수 = 수천만 스톤

## 3. 게임 모드 — Hybrid 본질

- 이동 / 미궁 층 진행
- 자원 (시간 / 물자) 소모
- 정수 흡수 / 합성
- 영구사망 / 부활 (★ 사제 / 야영지?)

### LLM 묘사 (★ 9B Q3)
- 백엔드 결과를 자연어로 묘사
- 캐릭터 대화
- 세계관 톤 유지

### LLM 가능성 평가 (★ 27B 또는 큰 모델)
- 자연어 지시 → 가능성 평가
- 특이 / 말도 안 되는 행동 → 어느 정도 가능성?
- 파티원 반응 (★ 따름 / 배신 / 임기응변)
- 일상생활 영향 (★ 특이 스탯)

### Frontend 시각화 (★ Phaser 4)
- 사이드뷰 sprite
- 캐릭터 (주인공 + 파티)
- 환경 (★ 왕궁 / 마을 / 미궁 / 균열 / 지하 / 바깥)
- 애니메이션 (★ 이동 / 공격 / 대화)
- HP / MP 바
- 인벤토리 / 도감 / 스킬 패널
