## 1. 게임 정체 — 새 정의

### 한 줄
> "작품 세계관에 빙의되어, 자연어로 지시하며, 사이드뷰 화면 + 채팅 묘사로 진행하는 시뮬 RPG"

### 핵심 흐름
```
1. 사용자 자연어 지시 (또는 선택지)
2. LLM이 가능성 평가 (★ "특이하고 말도 안 되는 선택지"도 평가)
3. 백엔드 시뮬 진짜:
   - HP/MP/스탯/위치 계산
   - 전투 / 이동 / 자원
   - 파티원 반응 (충성도/임기응변)
4. 사이드뷰 화면 갱신 (★ Phaser sprite)
5. 채팅에 묘사 (★ LLM)
6. 다음 턴 대기
```

| 항목 | AI Dungeon | Crack | FableAI | **WorldFork V2** |

## 3. 게임 모드 — Hybrid 본질

### 백엔드 시뮬 (★ 코드)

### LLM 묘사 (★ 9B Q3)

### LLM 가능성 평가 (★ 27B 또는 큰 모델)
