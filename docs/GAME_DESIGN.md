# WorldFork 게임 설계서 (GAME DESIGN DOCUMENT)

> 원작 『게임 속 바바리안으로 살아남기』(정윤강) 속 게임 **[던전 앤 스톤]**을 LLM 성향 엔진으로 재현·확장하는 게임의 완전 설계서.
>
> 핵심 정체성: **2D 픽셀 종스크롤 로그라이크 RPG + 성향 기반 자율 동료(LLM)**

> 작성: 2026-06-12 · 상태: 설계 v1 (원작 기반 확정)
> IP 정책: 게임 화면=원작 명칭(unmaskIp) / 코드·git=변환 명칭 / 원작 본문=.local(git 금지)

## 0. 한 문단 요약

플레이어는 게임 속 세계에 떨어진 존재로서 **바바리안 비요른**의 몸으로 미궁(던전)을 공략한다. 게임은 **2D 픽셀 종스크롤**이며, **죽으면 처음부터**(로그라이크/라이트는 §6에서 결정)인 하드코어 RPG다. 혼자서는 깊은 층을 못 가므로 **NPC 동료 파티가 필수**지만, 동료는 스크립트가 아니라 **성향(5축) 기반으로 자율 행동하고 플레이어 지시를 성향대로 해석(순응/변형/거부)하는 LLM 영혼**을 가진다. 미궁에서 **마석(화폐)과 장비**를 벌어 강해지고, **마을·성 등 거점**에서 자유롭게 행동(거래/팀 구성/의뢰/생활)하며, **세금·생존·더 깊은 층**이라는 압박 속에서 살아남는다. 원작 주인공이 "AI 판단이 멍청하다"고 불평했던 바로 그 지점 — **살아있는 동료** — 이 이 게임의 차별점이다.

## 1. 원작이 곧 설계서 — 핵심 추출

원작 1~30화에서 확인된 게임 [던전 앤 스톤]의 사실(게임 설계 근거):

| 원작 사실 | 게임 설계 함의 |
|---|---|
| 2D 픽셀 종스크롤, 인디 싱글 RPG | **비주얼 정체성 = 2D 픽셀** (ASCII 폐기) |
| 캐릭터 사망 시 처음부터 | **퍼머데스 / 로그라이크 긴장** |
| 9년 걸린 하드코어, 판단 미스=죽음 | **높은 난이도, 지식·판단 중시** |
| NPC 동료 게임 진행에 필수 | **파티 필수 = 성향 엔진의 무대** |
| "동료를 믿으면 안 된다"(6화) | **동료 불신·배신 가능성 = 긴장 축** |
| 종스크롤인데 자유도 높음 | **자유 행동 → LLM 판정(우리 엔진)** |
| "AI 판단 멍청"(1화 불평) | **우리 성향 동료가 그 답** (차별점) |
| 마석=화폐, 장비=아이템레벨 | **경제·성장 시스템** |
| 「」 시스템 메시지 | **게임 UI = 시스템 알림 (합쇼체)** |
| 마을/팀 비용/소모품/2층 도전(20화) | **거점·경제·층 진행 루프** |

핵심: 우리가 헤맸던 "핵심 루프·동기·긴장"은 전부 원작에 있다. 새로 발명할 게 아니라 **충실히 재현 + LLM 동료로 확장**한다.

## 2. 핵심 게임 루프

### 2.1 매크로 루프 (세션 단위)

```
[거점: 마을/성] ──→ [준비: 팀 구성·소모품·장비] ──→ [미궁 진입]
     ↑                                                      │
     │                                                      ↓
[보상 정산: 마석→돈, 장비]  ←──  [층 공략: 탐험·전투·사건]
     │                                                      │
     ↓                                              [더 깊은 층 or 귀환]
[성장·세금·생활·관계]                                       │
     └──────────────────── (사망 시: 로그라이크 리셋) ──────┘
```

### 2.2 마이크로 루프 (미궁 내 — RTwP)

```
실시간 진행(가변 속도/가속) — 파티가 성향대로 자율 이동/전투
        │
   [조우/분기점 발생]  → 자동 일시정지 옵션
        │
   플레이어 개입(일시정지) → 자연어 지시 → 동료 성향 해석(순응/변형/거부)
        │
   전투 해결(코드: HP/명중/피해/상태이상) — 판단이 생사 가름
        │
   결과 영구 반영(마석·장비·관계·세계 변화) → 재개
```

핵심: **개입 안 하면 빠르게 자동 진행**(속도 조절·가속), 개입할 때만 일시정지. 30초 턴 같은 건 없다.

## 3. 시스템 설계

### 3.1 캐릭터 (바바리안 비요른)

- **종족 특성(원작)**: HP 최고, 근력 기댓값 높음, 아다만티움 장비 착용 가능, "단순·멍청" 인식(연기로 활용 가능)
- **빌드 분기**: 대검 딜러(고화력·고위험) vs 탱커 육성(생존). 원작 주인공은 탱커 육성법으로 생존
- **스탯**: 근력/체력/민첩/지능 등 (원작 스탯 시스템 — §추후 수치 설계)
- **레벨/EXP**: 처치 시 EXP 획득(「고블린 처치 EXP +1」), 레벨업으로 성장
- **상태이상**: 출혈/중독 등 — 방치 시 사망(「생명력 5% 미만 경고」). **회복 타이밍이 생존의 핵심**

### 3.2 장비·아이템

- **아이템 레벨**: 장비마다 종합 아이템 레벨 기여(장화 +5, 양손망치 +30, 어깨보호대 +13)
- **장비 슬롯**: 무기/방어구/장화/보호대/배낭 등
- **배낭/인벤토리**: 배낭 착용 시 인벤토리 확장, 마법 배낭은 마석 자동 수집
- **소모품**: 회복약·횃불 등 미궁 준비물(돈으로 구매)

### 3.3 경제 (마석 = 화폐)

- **마석**: 몬스터 처치 시 드롭(9등급~상위), 등급별 가치. **게임 내 화폐**
- **수입**: 미궁 공략 → 마석 → 환전/거래
- **지출**: 소모품, 장비, **팀 구성 비용**(동료 고용), **세금**(거점 유지 압박)
- **긴장**: 돈이 부족하면 팀을 못 꾸리고 → 혼자 미궁 → 더 위험. 경제가 난이도와 직결

### 3.4 미궁 (던전) 구조

- **층(Floor)**: 1층부터 심층까지. 깊을수록 위험·보상↑ (원작 24화+ 층 언급)
- **1층**: 벽의 수정이 광원(횃불 불필요), 일부 암흑지대. 방향별 구역(원작: 동/서/남/북 다른 적)
- **적/몬스터**: 고블린·칼날늑대·구울·노움 등. 습성·약점 존재(지식으로 공략)
- **함정**: 고블린 덫 등 — 밟으면 피해(장비로 경감)
- **절차적 생성**: 던전 레이아웃·잡몹 배치는 PCG(반복 플레이 = 로그라이크)

### 3.5 거점 (마을·성)

- **마을**: 상점(장비·소모품), 길드(의뢰/팀), 환전(마석), 휴식
- **성**: 상위 거점 — 권력·정치·고급 의뢰(원작 후반 알현/권력 구조)
- **자유 행동**: 거점에서 거래/팀 구성/의뢰 수락/관계 형성/생활 — 던전만이 게임이 아님
- **세금**: 거점 유지·생활에 돈 필요 → 미궁 공략 동기

### 3.6 시스템 메시지 (게임 UI)

원작의 「」 합쇼체 시스템 알림을 그대로 채용:
- 「1층 수정 동굴에 입장했습니다.」 「고블린을 처치하였습니다. EXP +1」 「9등급 마석을 습득하였습니다.」 「[출혈]이 지속되고 있습니다.」 「경고: 생명력 5% 미만…」
- 게임 화면 = 시스템 메시지 + 픽셀 비주얼 + 서사(LLM, 결정적 순간)

## 4. 동료 시스템 — 차별점 (성향 엔진)

> 원작이 "동료 필수 + 믿으면 안 됨 + AI 판단 멍청"이라 한 그 지점을, **살아있는 성향 동료**로 답한다. 이미 구현된 V3 엔진(Phase 0-3)이 여기 붙는다.

### 4.1 성향 5축 (0-100)

충성 / 저돌 / 지혜 / 변덕 / 유대 — 각 동료의 영혼.

### 4.2 자율 행동 (평소, 코드 0토큰)

동료는 성향대로 알아서 움직인다(저돌↑→돌격, 지혜↑→정찰, 유대↑→구원). LLM 없이 코드 패턴 — 비용 0, 즉각.

### 4.3 지시 해석 (개입 시, LLM)

플레이어 자연어 지시 → 성향×지시×상황 → **순응/변형/거부 + 근거**. 충성↑ 순응, 지혜↑ 변형, 위험 간파 시 거부, 변덕↑ 예측 불가.

### 4.4 불신·배신 (원작 핵심)

- 동료는 무조건 아군이 아니다. **신뢰 관계(relationship -100~100)**에 따라 배신·이탈 가능(원작 "믿으면 안 됨")
- 팀 구성 시 비용·성향·신뢰를 따져야 하는 전략적 선택
- 배신은 영구 기록(npc memory) → 세계가 기억

### 4.5 LLM 빈도 (비용)

평소 코드, 분기점(새 적/위기/갈등/개입)만 LLM. 파티 3-4명이어도 비용 폭발 없음(검증됨).

---

## 5. 비주얼 — 2D 픽셀 (정체성)

> ASCII 네모는 임시였고 폐기. 원작이 2D 픽셀이며, 우리는 비요른 LoRA·ComfyUI·Flux 자산을 이미 보유.

- **종스크롤 2D 픽셀**: 던전·캐릭터·적·아이템 픽셀 아트
- **비요른 LoRA**: 주인공·핵심 캐릭터 일러스트(loss 0.1796 학습 완료)
- **ComfyUI + Flux**: 배경·에셋 생성 파이프라인(출력 경로 기존 설정)
- **UI 오버레이**: HTML/CSS UI + 픽셀 배경 합성(Phase 6a 목업 진행했던 방향)
- **로드맵**: 웹 단계는 픽셀 스프라이트/타일 렌더로 복원 → Godot 이주 시 네이티브 2D

## 6. 장르 결정 — 하드코어 하이브리드 (★ 확정)

> 결정(2026-06-12, 사용자): **캐릭터 퍼머데스(하드코어) + 세계·관계 영속**. 원작의 죽음 무게와 성향 동료 엔진의 관계 가치를 둘 다 살린다. 순수 로그라이크는 매 사망 시 관계·npc memory·세계 flags(Phase 2)를 리셋해 차별점을 스스로 깎고 LLM 비용도 반복되므로 기각.

### 6.1 사망 시 소멸 (캐릭터 귀속)

- 빌드: 스탯·스킬·**흡수한 정수 능력**
- 장비·아이템 레벨·인벤토리, 들고 있던 마석/현금
- 현재 미궁 진행·깊이
- 제도적 직위(남작 작위 등) — 캐릭터 귀속이라 소멸. 새 캐릭터는 다시 증명(단 이전 평판 덕에 재기는 빠름)

### 6.2 영속 (플레이어 귀속 — 죽음을 넘음)

- **동료 관계(relationship)**, **npc memory**(약속·배신·사건)
- **세계 flags**, 메타 지식/해금(맵·레시피·시스템 이해)
- 사회적 평판
- **동료 생존/사망 명부**

### 6.3 동료 영구 사망 (원작식 무게)

- 런 중 동료는 **영구 사망 가능** → 명부에서 영구 제거, 부활 없음
- **배신도 영구 기록** → 세계가 기억. 죽음·배신의 무게가 원작 정서

### 6.4 정합성 — 관계는 플레이어에 묶인다

- 플레이어 캐릭터가 죽어도 동료는(생존 시) 다음 런으로 이어진다. 관계가 **캐릭터가 아니라 플레이어 정체성**에 귀속.
- 원작 메타로 정당화: **빙의·관리자 기록·다중 플레이어** → "육신은 갈려도 너라는 존재를 세계가 기억한다." 2회차 동료 재회("또 너냐")가 자연스럽고, 메타 떡밥을 메커닉으로 회수.

### 6.5 구현 함의 (세이브 구조)

```
PER-RUN  (캐릭터 세이브, 사망 시 폐기)
└── 스탯·스킬·정수능력 / 장비·인벤·소지금 / 미궁 진행 / 직위

PERSISTENT (월드 세이브, 죽음을 넘어 유지)
└── relationship / npc memory / world flags / 해금·지식 / 평판 / 동료 생사 명부
```

→ 핵심 루프(§2)의 사망 처리 = "캐릭터 세이브 폐기 + 월드 세이브 유지 + 신 캐릭터 생성". 동료 사망 = 월드 세이브의 명부 갱신(영구).

## 7. 플레이어 동기 (원작 기반)

1. **생존** — 죽으면 끝(또는 큰 손실). 매 전투가 진검승부
2. **강해짐** — 레벨·장비·빌드로 더 깊은 층 도전
3. **돈** — 마석 벌어 팀·장비·세금 충당. 경제가 생존과 직결
4. **더 깊이** — 미궁 심층의 보상·비밀(원작 「Gate of the Abyss」 최종)
5. **관계·서사** — 동료와의 신뢰/배신, 거점 NPC, 세계의 변화(영구)
6. **탈출/진실** — (원작 메타) 게임 세계에 갇힌 자의 생존과 진실 추구

## 8. 긴장 구조

- **죽음의 무게**: 퍼머데스(또는 큰 페널티) → 모든 선택에 무게
- **자원 압박**: 마석·소모품·세금 → 무리한 도전 vs 안전
- **동료 불확실성**: 성향대로 안 따를 수 있고, 배신도 가능 → 통제 불완전성이 긴장이자 드라마
- **지식 vs 미지**: 아는 몬스터는 공략, 새 위협은 도박
- **깊이의 유혹**: 더 내려갈수록 보상↑ 위험↑ — 욕심과 신중의 줄타기

## 9. 기술 아키텍처 (현 자산 연결)

```
게임 코어 (코드 — 즉각·결정적)
├── 전투: HP/명중/피해/상태이상 (기존 combat 로직)
```
