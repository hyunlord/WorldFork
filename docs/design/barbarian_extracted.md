## 1. 작품 본질 (★ 나무위키 1.개요 / 2.줄거리 / 5.특징)

- **장르**: 한국 게임빙의물, 미궁물 웹소설 (게임빙의 + 던전물 + 야만전사물 표면 + 던전물 본질)
- **작가**: 정윤강 (전작: 〈내가 믿는 것은 검〉, 〈알고보니 일대일 천재〉)
- **연재**: 네이버 시리즈 2021-03-31 ~ 연재 중 (★ 진행 중)
- **번역**: Surviving the Game as a Barbarian (욘더, 영어)
- **줄거리** (★ 인용):
  > "2D 버전으로도 못 깼는데, 이걸 현실에서 깨라고?"
  > "어쩌면 평생 여기서 살아야 할지도 모르겠다."
  > 인디 게임 **던전 앤 스톤(Dungeon & Stone)** 의 바바리안에 빙의된 주인공이, 죽음이 가득한 미궁 속을 탐험하며 살아남기 위해 고군분투하는 이야기

## 10. 본 시뮬레이션 게임 디자인 본질 (★ 자료에서 진짜 추론)

본 작품은 **수치 시스템 + 미궁 깊이 + 영구사망 + 파티** 의 4중 핵심으로:
1. 캐릭터 능력은 정수/종족/재능/외피 등 **다중 자원** 으로 표현
2. 미궁은 1~8+층 + 균열의 **수직 깊이 + 횡적 위험도** 표현
3. 영구사망 + 부활(?) — **위험 vs 보상** 의 진짜 긴장
4. 파티 클래스 다양성 (전사/사제/연금술사/탐험가) — **상호의존**

**RPG적 요소 진짜 본질** (본인 의문 정합):
- HP / 방어력 (외피 + 정령화 + 내성 수치)
- 인벤토리 + 장비 (★ 무기/방어구/소비/기타)
- 스킬 (★ Q/W/E/R 4개 + 작품 추가)
- 도감 (★ TAB — 몬스터/아이템/지역?)
- 레벨업 + 정수 / 종족 특성 숙련
- 동료 (★ 에르웬 등 + 부활)
- 위치 (★ 미궁 층/방/균열)
