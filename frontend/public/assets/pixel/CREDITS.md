# 던전 픽셀 타일 — 라이선스/출처

이 디렉터리의 16x16 픽셀 타일(floor/wall/player/enemy/npc/item/stair/door)은
**WorldFork 프로젝트가 직접 디자인한 원작 자산**이며 **CC0 1.0 (퍼블릭 도메인)**
으로 배포한다 — 자유 사용·수정·재배포 가능, 출처 표기 불요.

- 생성기: `tools/gen_pixel_tiles.py` (순수 stdlib, 외부 패키지/다운로드 0)
- 재생성: `python tools/gen_pixel_tiles.py`

## 외부 에셋을 더할 때 (★ 라이선스 필수)

타일 매핑은 에셋 무관이라 더 정교한 팩으로 PNG만 교체하면 된다. 외부 자산은
**CC0 또는 CC-BY만** 사용하고, 아래에 출처/라이선스를 명시한다(불명/제한 금지).
